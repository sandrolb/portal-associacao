# Quickstart: Portal Basquete

## Opening the Portal
1. Open `index.html` in a modern browser.
2. Alternatively, serve the project folder with a local static server (e.g. VS Code Live Server or `python -m http.server`) and browse to `http://localhost:8000`.

## Core Features

- Top banner with the association logo and horizontal navigation.
- Sections for Quem Somos, História, Categorias, Campeonatos, Títulos Conquistados, Notícias, Links para redes, Contatos and Inscrição.
- Light/dark theme toggle with white-based light mode and navy-blue dark mode.
- Athlete registration form saved to browser Local Storage.

## How to Use

1. Click navigation links to jump between sections.
2. Use the theme toggle button to switch between light and dark modes.
3. Fill the registration form with athlete details and click submit.
4. After submission, verify stored entries in browser Local Storage if needed.

## Validation

- Confirm the portal displays correctly on mobile, tablet and desktop widths.
- Ensure the registration form validates required fields and displays inline feedback.
- Confirm the form data persists in Local Storage after submission.
